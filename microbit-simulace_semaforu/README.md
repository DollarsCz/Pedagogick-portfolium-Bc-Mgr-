# Simulace semaforu pomocí Micro:bitu (didaktická část)

Tato složka obsahuje materiály pro realizaci úlohy „Simulace semaforu“ ve výuce.

## Soubory
- **zadani-pro-zaky.md** – instrukce a zadání pro žáky
- **metodicky-komentar.md** – cíle, pomůcky, průběh hodiny, diferenciace a hodnocení

## Doporučení
Úlohu lze realizovat samostatně nebo ve dvojicích. Doporučuje se krátká úvodní demonstrace polarity LED a role rezistoru.
